 **So, why Koffer?**

 Koffer is a small project aimed to bring back useability to mainly the Windows Phone platform. If you are from the communities, you likely know that Microsoft have been removing abilities from Windows Mobile. Whilst this is mainly to get people to let the platform die, many won't leave it. With this project, we aim to bring back data syncronisation and cloud storage access with the help of koofr.eu.
